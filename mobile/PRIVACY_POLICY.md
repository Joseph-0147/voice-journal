# Echo Privacy Policy

Last updated: April 7, 2026

## Overview
Echo ("we", "our", "us") provides a voice journaling app that helps users record audio entries and generate insights. This policy explains how we collect, use, and protect information.

## Data We Collect
- Account information: email address, display name, hashed password
- Journal content: audio recordings, transcripts, generated insights
- Technical data: device type, app version, crash diagnostics

## How We Use Data
- Provide core journaling features
- Generate AI insights for your entries
- Sync data across devices
- Improve reliability and performance

## AI Processing
Audio and transcript content may be processed by our server and trusted AI service providers for transcription and insight generation.

## Storage and Security
- Data is encrypted in transit using HTTPS
- Access to user data requires authentication
- Audio files are stored in secure cloud object storage

## Data Sharing
We do not sell personal data. We only share data with service providers needed to deliver app functionality (for example cloud storage and AI processing).

## Your Choices
You can request account deletion and data removal by contacting support.

## Children
Echo is not directed to children under 13.

## Contact
For privacy requests, contact: support@echojournal.app
